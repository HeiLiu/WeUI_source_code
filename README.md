## 概述  
WEUI是一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信web开发量身设计，可以令用户的使用感知更加统一。包含`button`、`cell`、`dialog`、`progress`、`toast`、`actionsheet`、`icon`、等各式元素
